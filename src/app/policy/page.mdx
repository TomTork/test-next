# Политика конфиденциальности
## Общие положения
- Typescript-Приложение - это особо организованное приложение, представляющее собой веб-сайт, написанный на этом языке программирования;
- Фреймворк - это библиотека в виде движка, позволяющая быстро писать Typescript-Приложения;
- Next.js - это фреймворк для быстрого написания веб-сайтов.
